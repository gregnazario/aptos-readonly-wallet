View-Only Wallet

Impersonates an Aptos address. Transactions are never signed — the payload is logged and (in Ask-me mode) shown in an approval window so you can accept, reject, or copy it to execute elsewhere.

Ask me opens a window with the parsed payload + Accept/Reject. Simulate accept returns a fake APPROVED (all-zero signatures, invalid on-chain). Reject always returns a user-rejection.
